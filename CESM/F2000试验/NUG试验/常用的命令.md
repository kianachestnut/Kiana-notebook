
# 🚀 **Linux 常用命令超级小手册（科研服务器版）**

## ------------------------------------

# 🗂 文件与文件夹操作

### **查看当前路径**

```
pwd
```

### **查看文件与权限**

```
ls -l         # 显示详细信息
ls -a         # 显示包含隐藏文件
ls -lh        # 人类可读大小
```

### **进入目录**

```
cd dirname
cd ..         # 返回上一级
cd ~          # 返回 home
```

### **创建文件/文件夹**

```
mkdir dir
mkdir -p a/b/c   # 递归创建
touch file.txt   # 创建空文件
```

### **复制**

```
cp file1 file2
cp -r dir1 dir2   # 复制整个目录
```

### **移动/重命名**

```
mv old new
```

### **删除**

⚠️ 不可逆，请务必确认路径

```
rm file
rm -r dir
rm -rf dir       # 强制删除（删除不掉时的终极命令）
```

---

# 🔐 权限相关（SFTP error #4 常见原因）

### **查看权限**

```
ls -l
```

### **给自己权限**

```
chmod u+rwx file_or_dir
chmod -R u+rwx dir
```

### **修改所有者（如果你有 sudo 权限）**

```
sudo chown -R user:group dir
```

---

# 📦 压缩与解压

### **tar 文件**

```
tar -czvf a.tar.gz dir/      # 压缩
tar -xzvf a.tar.gz           # 解压
```

### **zip**

```
zip -r a.zip dir/
unzip a.zip
```

---

# 📥 文件传输

### **scp 传文件到远程**

```
scp file user@ip:/path/
```

### **scp 下载**

```
scp user@ip:/path/file .
```

# 📑 查看文件内容

```

tail file

```

# 🔎 查找

### **查找文件名**

```
find /path -name "*.nc"
```

# 🧠 进程管理（跑 CESM、WPS、WRF 最常见）

### **查看进程**

```
top
htop              # 如果服务器装了更友好的
ps aux | grep xxx # 搜索与 xxx 相关的进程
```

### **杀进程**

```
kill PID
kill -9 PID       # 强制杀
```
